# Primitive Element Theorem

The Primitive Element Theorem guarantees that many field extensions that appear to require multiple generators can in fact be generated by a single element. This simplifies the structure theory of field extensions significantly.

---

## Statement

<Theorem title="Primitive Element Theorem" number="5.22">
Let $E/F$ be a finite separable extension. Then $E = F(\alpha)$ for some $\alpha \in E$.

In other words, every finite separable extension is a **simple extension**.
</Theorem>

<Remark title="Characteristic zero">
Since every extension in characteristic $0$ is separable, the Primitive Element Theorem applies to all finite extensions of $\mathbb{Q}$, $\mathbb{R}$, or any number field.
</Remark>

---

## Proof sketch

<Proof>
We prove the case when $F$ is infinite (the finite field case is separate and easier).

**Finite field case**: If $F$ is finite, then $E$ is also finite, so $E^\times$ is cyclic. Any generator $\alpha$ of $E^\times$ gives $E = F(\alpha)$.

**Infinite field case**: It suffices to show $F(\alpha, \beta) = F(\gamma)$ for some $\gamma$ (then induct on the number of generators). Consider $\gamma = \alpha + c\beta$ for $c \in F$. We need to choose $c$ so that $F(\gamma) = F(\alpha, \beta)$.

Let $f(x) = \min_F(\alpha)$ and $g(x) = \min_F(\beta)$. Consider the polynomial $h(x) = f(\gamma - cx) \in F(\gamma)[x]$. Then $h(\beta) = f(\alpha) = 0$, so $\beta$ is a common root of $h(x)$ and $g(x)$ in $E$.

If $c$ is chosen so that $\beta$ is the **only** common root of $h$ and $g$, then $\gcd(h, g)$ computed in $F(\gamma)[x]$ has $\beta$ as its only root. Since $g$ is separable, this forces $\beta \in F(\gamma)$, and then $\alpha = \gamma - c\beta \in F(\gamma)$, giving $F(\alpha,\beta) = F(\gamma)$.

The condition fails only for finitely many values of $c$ (each "bad" $c$ comes from a pair $(\alpha_i, \beta_j)$ of roots). Since $F$ is infinite, a suitable $c$ exists.
</Proof>

## Examples

<Example id="pet-sqrt2-sqrt3" title="Primitive element for Q(sqrt(2), sqrt(3))">
$\mathbb{Q}(\sqrt{2}, \sqrt{3}) = \mathbb{Q}(\sqrt{2} + \sqrt{3})$. To verify: let $\gamma = \sqrt{2} + \sqrt{3}$. Then:

$$\gamma^2 = 5 + 2\sqrt{6}, \quad \text{so } \sqrt{6} = \frac{\gamma^2 - 5}{2} \in \mathbb{Q}(\gamma).$$

$$\gamma \cdot \sqrt{6} = \sqrt{12} + \sqrt{18} = 2\sqrt{3} + 3\sqrt{2} \in \mathbb{Q}(\gamma).$$

$$\sqrt{2} = \gamma \cdot \sqrt{6} - 2\gamma = 3\sqrt{2} + 2\sqrt{3} - 2\sqrt{2} - 2\sqrt{3} = \sqrt{2}.$$

More directly: $\sqrt{2} = \frac{\gamma\sqrt{6} - 2\gamma}{1}$... Let us use a cleaner argument. We have $\gamma^2 = 5 + 2\sqrt{6}$, so $\sqrt{6} \in \mathbb{Q}(\gamma)$. Then $\sqrt{3} = \sqrt{6}/\sqrt{2}$ and $\sqrt{2} = \gamma - \sqrt{3}$. So $\sqrt{2} \in \mathbb{Q}(\gamma)$ as long as $\sqrt{3} \in \mathbb{Q}(\gamma)$. Since $\gamma\sqrt{6} = \sqrt{12} + \sqrt{18} = 2\sqrt{3} + 3\sqrt{2}$ and $2\gamma = 2\sqrt{2} + 2\sqrt{3}$, we get $\gamma\sqrt{6} - 2\gamma = \sqrt{2}$, so $\sqrt{2} \in \mathbb{Q}(\gamma)$, hence $\sqrt{3} = \gamma - \sqrt{2} \in \mathbb{Q}(\gamma)$.

The minimal polynomial of $\gamma$ over $\mathbb{Q}$ is $x^4 - 10x^2 + 1$.
</Example>

<Example id="pet-i-sqrt2" title="Primitive element for Q(i, sqrt(2))">
$\mathbb{Q}(i, \sqrt{2}) = \mathbb{Q}(i + \sqrt{2})$. Let $\gamma = i + \sqrt{2}$. Then:

$$\gamma^2 = -1 + 2 + 2i\sqrt{2} = 1 + 2i\sqrt{2}$$

$$\gamma^2 - 1 = 2i\sqrt{2}, \quad (\gamma^2-1)^2 = -8, \quad \gamma^4 - 2\gamma^2 + 9 = 0.$$

From $\gamma^2 - 1 = 2i\sqrt{2}$, we get $i\sqrt{2} \in \mathbb{Q}(\gamma)$. Then $\sqrt{2} = \gamma - i$ requires $i$, and $i = (\gamma^2-1)/(2\sqrt{2})$ requires $\sqrt{2}$. Using $(\gamma^2-1)^2 = -8$ gives $(\gamma^2-1)^2/8 = -1$, so $i = (\gamma^2-1)/(2\sqrt{2})$. We also have $(\gamma^2-1)^3 = -8(\gamma^2-1) = -16i\sqrt{2}$, so this works out to recover both $i$ and $\sqrt{2}$.

The minimal polynomial is $x^4 - 2x^2 + 9$.
</Example>

<Example id="pet-cuberoot-omega" title="Primitive element for Q(cbrt(2), omega)">
$\mathbb{Q}(\sqrt[3]{2}, \omega) = \mathbb{Q}(\sqrt[3]{2} + \omega)$. This is a degree $6$ extension, and a single primitive element generates it. The minimal polynomial of $\sqrt[3]{2} + \omega$ has degree $6$.
</Example>

<Example id="pet-finite-field" title="Primitive element for finite fields">
$\mathbb{F}_{p^n}^\times$ is cyclic (by the theorem on multiplicative groups of finite fields), so any generator $\alpha$ of $\mathbb{F}_{p^n}^\times$ satisfies $\mathbb{F}_{p^n} = \mathbb{F}_p(\alpha)$.

For example, in $\mathbb{F}_{4} = \mathbb{F}_2(\alpha)$ where $\alpha^2 + \alpha + 1 = 0$, the element $\alpha$ generates the whole field.
</Example>

## When the theorem fails

<Example id="pet-fails-inseparable" title="Failure for inseparable extensions">
Let $F = \mathbb{F}_p(s, t)$ (rational functions in two variables over $\mathbb{F}_p$) and $E = F(\sqrt[p]{s}, \sqrt[p]{t})$. Then $[E:F] = p^2$, but $E \neq F(\alpha)$ for any single $\alpha$.

Indeed, every $\alpha \in E$ satisfies $\alpha^p \in F$ (since the Frobenius map on $E$ lands in $F$), so $[F(\alpha):F] \le p < p^2 = [E:F]$.

This extension is **purely inseparable**: every element satisfies $\alpha^p \in F$.
</Example>

<Example id="pet-inseparable-char2" title="An inseparable extension in characteristic 2">
Let $F = \mathbb{F}_2(t)$ and $E = F(\sqrt{t}) = F[x]/(x^2 - t)$. This is inseparable since $x^2 - t = (x - \sqrt{t})^2$. The extension has degree $2$, and $E = F(\sqrt{t})$ is simple, but it is not separable.

Now consider $E = \mathbb{F}_2(s, t)$ and $K = \mathbb{F}_2(s^2, t^2)$. Then $[E:K] = 4$, and $E$ cannot be written as $K(\alpha)$ for any single $\alpha$ (by the argument above: $\alpha^2 \in K$ for all $\alpha \in E$).
</Example>

## Finding primitive elements

<Example id="find-pet-linear-combo" title="Linear combination method">
Given $E = F(\alpha, \beta)$, the proof shows $\gamma = \alpha + c\beta$ works for all but finitely many $c \in F$. In practice:
- Try $c = 1$: check if $F(\alpha + \beta) = F(\alpha, \beta)$.
- If not, try $c = 2, 3, \ldots$.
- Over $\mathbb{Q}$, the choice $c = 1$ almost always works.
</Example>

<Example id="find-pet-degree-check" title="Degree check method">
To verify $\gamma$ is primitive for $E/F$: compute $\min_F(\gamma)$. If $\deg \min_F(\gamma) = [E:F]$, then $F(\gamma) = E$.

For $\gamma = \sqrt{2} + \sqrt{3}$: $\gamma$ satisfies $x^4 - 10x^2 + 1 = 0$. If this polynomial is irreducible over $\mathbb{Q}$ (check by the rational root theorem and irreducibility tests), then $[\mathbb{Q}(\gamma):\mathbb{Q}] = 4 = [\mathbb{Q}(\sqrt{2},\sqrt{3}):\mathbb{Q}]$, confirming $\gamma$ is primitive.
</Example>

<Example id="pet-general-quadratic" title="Primitive element for any two quadratic extensions">
If $K_1 = F(\sqrt{a})$ and $K_2 = F(\sqrt{b})$ with $a/b$ not a perfect square in $F$, then $K_1 K_2 = F(\sqrt{a} + \sqrt{b})$.

This follows since $(\sqrt{a}+\sqrt{b})^2 = a + b + 2\sqrt{ab}$, so $\sqrt{ab} \in F(\sqrt{a}+\sqrt{b})$, and then $\sqrt{a} = (\sqrt{a}+\sqrt{b}) \cdot \sqrt{ab} / (something)$... More carefully: $(\sqrt{a}+\sqrt{b})\sqrt{ab} = a\sqrt{b} + b\sqrt{a}$, and with $\sqrt{a}+\sqrt{b}$ and $a\sqrt{b}+b\sqrt{a}$ we recover $\sqrt{a}$ and $\sqrt{b}$ by linear algebra (since $a \neq b$).
</Example>

## Consequences

<Theorem title="Finite number of intermediate fields" number="5.23">
If $E/F$ is a finite separable extension, then there are finitely many intermediate fields $F \subseteq K \subseteq E$.
</Theorem>

<Proof>
By the Primitive Element Theorem, $E = F(\alpha)$. Every intermediate field $K$ satisfies $E = K(\alpha)$, so $\min_K(\alpha)$ divides $\min_F(\alpha)$ in $E[x]$. Since $\min_F(\alpha)$ has finitely many monic divisors, there are finitely many possibilities for $\min_K(\alpha)$, and $K$ is determined by $\min_K(\alpha)$.
</Proof>

<Example id="converse-fails" title="Infinitely many intermediate fields imply inseparability">
Conversely, if $E/F$ has infinitely many intermediate fields, then $E/F$ is not separable (and in particular, $E$ is not simple over $F$). The extension $\mathbb{F}_p(s,t)/\mathbb{F}_p(s^p, t^p)$ has infinitely many intermediate fields: $\mathbb{F}_p(s^p, t^p, as + bt)$ for each $[a:b] \in \mathbb{P}^1(\mathbb{F}_p)$... in fact, for each element in $\mathbb{F}_p(s^p,t^p)$, giving uncountably many.
</Example>

<Example id="pet-number-fields" title="Number fields are simple extensions">
Every number field $K$ (finite extension of $\mathbb{Q}$) is a simple extension: $K = \mathbb{Q}(\alpha)$ for some algebraic number $\alpha$. This is because $\mathbb{Q}$ has characteristic $0$, so all extensions are separable.

For example, $\mathbb{Q}(\sqrt{2}, \sqrt{3}, \sqrt{5}) = \mathbb{Q}(\sqrt{2}+\sqrt{3}+\sqrt{5})$ is generated by a single element.
</Example>

## Summary

| Extension | Separable? | Primitive element | Minimal polynomial |
|---|---|---|---|
| $\mathbb{Q}(\sqrt{2})/\mathbb{Q}$ | Yes | $\sqrt{2}$ | $x^2 - 2$ |
| $\mathbb{Q}(\sqrt{2},\sqrt{3})/\mathbb{Q}$ | Yes | $\sqrt{2}+\sqrt{3}$ | $x^4-10x^2+1$ |
| $\mathbb{Q}(i,\sqrt{2})/\mathbb{Q}$ | Yes | $i + \sqrt{2}$ | $x^4-2x^2+9$ |
| $\mathbb{F}_{p^n}/\mathbb{F}_p$ | Yes | generator of $\mathbb{F}_{p^n}^\times$ | irreducible of degree $n$ |
| $\mathbb{F}_p(s^{1/p},t^{1/p})/\mathbb{F}_p(s,t)$ | **No** | does not exist | N/A |

<Remark title="Looking ahead">
The Primitive Element Theorem, combined with the [Fundamental Theorem of Galois Theory](/modern-algebra/ch05-fields-galois/theorems/t01-fundamental-galois), shows that every Galois extension of $\mathbb{Q}$ is generated by a single algebraic number whose symmetries are captured by the Galois group.
</Remark>
