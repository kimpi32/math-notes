# 수렴과 발산 (Convergence and Divergence)

The epsilon-N definition of convergence is the cornerstone of rigorous analysis. Every subsequent definition in this subject (limits of functions, continuity, differentiability) is modeled on this pattern.

---

## The epsilon-N definition

<Definition title="Convergence of a sequence" number="2.5">
A sequence $(a_n)$ **converges** to a limit $L \in \mathbb{R}$ if:

$$\forall \varepsilon > 0, \quad \exists N \in \mathbb{N}, \quad \forall n \geq N : \quad |a_n - L| < \varepsilon.$$

We write $\lim_{n \to \infty} a_n = L$ or $a_n \to L$.

A sequence that does not converge is said to **diverge**.
</Definition>

<Remark>
The order of quantifiers is crucial. The definition says: for **every** tolerance $\varepsilon > 0$, no matter how small, there is **some** stage $N$ beyond which **all** terms $a_n$ lie within $\varepsilon$ of $L$. The value $N$ may (and typically does) depend on $\varepsilon$.
</Remark>

---

## Fundamental examples

<Example title="1/n converges to 0">
**Claim**: $\lim_{n \to \infty} \frac{1}{n} = 0$.

**Proof**: Let $\varepsilon > 0$. By the Archimedean property, there exists $N \in \mathbb{N}$ with $N > \frac{1}{\varepsilon}$. For all $n \geq N$:

$$\left|\frac{1}{n} - 0\right| = \frac{1}{n} \leq \frac{1}{N} < \varepsilon.$$
</Example>

<Example title="Constant sequence">
**Claim**: If $a_n = c$ for all $n$, then $\lim_{n \to \infty} a_n = c$.

**Proof**: Let $\varepsilon > 0$. Choose $N = 1$. For all $n \geq 1$: $|a_n - c| = |c - c| = 0 < \varepsilon$.
</Example>

<Example title="n/(n+1) converges to 1">
**Claim**: $\lim_{n \to \infty} \frac{n}{n+1} = 1$.

**Proof**: Let $\varepsilon > 0$. We have $\left|\frac{n}{n+1} - 1\right| = \frac{1}{n+1} < \frac{1}{n}$. Choose $N > \frac{1}{\varepsilon}$. Then for $n \geq N$: $\left|\frac{n}{n+1} - 1\right| < \frac{1}{n} \leq \frac{1}{N} < \varepsilon$.
</Example>

<Example title="(2n+1)/(3n+5) converges to 2/3">
**Claim**: $\lim_{n \to \infty} \frac{2n+1}{3n+5} = \frac{2}{3}$.

**Proof**: We compute $\left|\frac{2n+1}{3n+5} - \frac{2}{3}\right| = \left|\frac{3(2n+1) - 2(3n+5)}{3(3n+5)}\right| = \frac{7}{3(3n+5)} < \frac{7}{9n}$.

Given $\varepsilon > 0$, choose $N > \frac{7}{9\varepsilon}$. Then for $n \geq N$: $\left|\frac{2n+1}{3n+5} - \frac{2}{3}\right| < \frac{7}{9n} \leq \frac{7}{9N} < \varepsilon$.
</Example>

## Uniqueness of limits

<Theorem title="Uniqueness of limits for sequences">
If $a_n \to L$ and $a_n \to L'$, then $L = L'$.
</Theorem>

<Proof>
Suppose $L \neq L'$. Let $\varepsilon = \frac{|L - L'|}{2} > 0$. Since $a_n \to L$, there exists $N_1$ with $|a_n - L| < \varepsilon$ for $n \geq N_1$. Since $a_n \to L'$, there exists $N_2$ with $|a_n - L'| < \varepsilon$ for $n \geq N_2$. For $n \geq \max(N_1, N_2)$:

$$|L - L'| \leq |L - a_n| + |a_n - L'| < \varepsilon + \varepsilon = |L - L'|,$$

a contradiction.
</Proof>

## Divergence

<Definition title="Divergence to infinity" number="2.6">
A sequence $(a_n)$ **diverges to** $+\infty$ (written $a_n \to +\infty$) if:

$$\forall M > 0, \quad \exists N \in \mathbb{N}, \quad \forall n \geq N : \quad a_n > M.$$

Similarly, $a_n \to -\infty$ if for every $M < 0$, there exists $N$ with $a_n < M$ for all $n \geq N$.
</Definition>

<Example title="n squared diverges to infinity">
**Claim**: $a_n = n^2 \to +\infty$.

**Proof**: Given $M > 0$, choose $N > \sqrt{M}$. For $n \geq N$: $a_n = n^2 \geq N^2 > M$.
</Example>

<Example title="Oscillating divergence">
The sequence $a_n = (-1)^n$ diverges, but it does **not** diverge to $\pm \infty$. It oscillates between $-1$ and $1$.

**Proof of divergence**: Suppose $a_n \to L$. For $\varepsilon = \frac{1}{2}$, there exists $N$ with $|a_n - L| < \frac{1}{2}$ for all $n \geq N$. But then $|a_N - a_{N+1}| \leq |a_N - L| + |L - a_{N+1}| < 1$. However $|a_N - a_{N+1}| = |(-1)^N - (-1)^{N+1}| = 2$, a contradiction.
</Example>

## Algebra of limits

<Theorem title="Algebraic limit theorem">
Let $a_n \to A$ and $b_n \to B$. Then:
1. $a_n + b_n \to A + B$.
2. $c \cdot a_n \to c \cdot A$ for any $c \in \mathbb{R}$.
3. $a_n \cdot b_n \to A \cdot B$.
4. $a_n / b_n \to A / B$, provided $B \neq 0$ and $b_n \neq 0$ for all $n$.
</Theorem>

<Example title="Using the algebraic limit theorem">
7. $\lim \frac{3n^2 + n}{2n^2 - 1} = \lim \frac{3 + 1/n}{2 - 1/n^2} = \frac{3 + 0}{2 - 0} = \frac{3}{2}$.
8. $\lim \frac{1}{n^2} = \lim \frac{1}{n} \cdot \frac{1}{n} = 0 \cdot 0 = 0$.
</Example>

## Convergent sequences are bounded

<Theorem title="Convergent implies bounded">
If $(a_n)$ converges, then $(a_n)$ is bounded.
</Theorem>

<Proof>
Let $a_n \to L$. Choose $\varepsilon = 1$. There exists $N$ with $|a_n - L| < 1$ for all $n \geq N$, so $|a_n| < |L| + 1$ for $n \geq N$. Set $K = \max(|a_1|, |a_2|, \ldots, |a_{N-1}|, |L| + 1)$. Then $|a_n| \leq K$ for all $n$.
</Proof>

<Remark>
The converse is false: $a_n = (-1)^n$ is bounded but does not converge.
</Remark>

## Order limit theorem

<Theorem title="Order limit theorem">
Let $a_n \to A$ and $b_n \to B$.
1. If $a_n \leq b_n$ for all $n \geq N_0$ (some $N_0$), then $A \leq B$.
2. If $A < B$, then there exists $N$ with $a_n < b_n$ for all $n \geq N$.
</Theorem>

<Example title="Strictness can be lost">
9. Let $a_n = 0$ and $b_n = \frac{1}{n}$. Then $a_n < b_n$ strictly for all $n$, but $\lim a_n = 0 = \lim b_n$. Strict inequality can become non-strict in the limit.
</Example>

---

## Squeeze theorem for sequences

<Theorem title="Squeeze theorem">
If $a_n \leq c_n \leq b_n$ for all $n \geq N_0$ and $\lim a_n = \lim b_n = L$, then $\lim c_n = L$.
</Theorem>

<Example title="Squeeze theorem application">
10. $\lim \frac{\sin n}{n} = 0$: Since $-1 \leq \sin n \leq 1$, we have $\frac{-1}{n} \leq \frac{\sin n}{n} \leq \frac{1}{n}$. Both bounds converge to $0$, so $\frac{\sin n}{n} \to 0$.
</Example>

<Example title="Squeeze with geometric decay">
11. $\lim \frac{n}{2^n} = 0$: For $n \geq 3$, $2^n = (1 + 1)^n \geq \frac{n(n-1)}{2}$ by the binomial theorem, so $0 \leq \frac{n}{2^n} \leq \frac{2}{n-1} \to 0$.
</Example>

<Example title="Squeeze with n-th root">
12. $\lim n^{1/n} = 1$: Write $n^{1/n} = 1 + h_n$ where $h_n \geq 0$ for $n \geq 1$. Then $n = (1 + h_n)^n \geq \frac{n(n-1)}{2} h_n^2$ for $n \geq 2$, so $h_n^2 \leq \frac{2}{n-1}$, giving $0 \leq h_n \leq \sqrt{\frac{2}{n-1}} \to 0$.
</Example>
