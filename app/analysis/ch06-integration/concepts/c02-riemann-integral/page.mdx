# Riemann Integral

The Riemann integral gives a precise meaning to "the area under a curve." A bounded function is Riemann integrable when its upper and lower integrals coincide.

---

## Definition

<Definition title="Riemann integrable function" number="6.6">
A bounded function $f: [a,b] \to \mathbb{R}$ is **Riemann integrable** on $[a,b]$ if

$$\underline{\int_a^b} f = \overline{\int_a^b} f.$$

When this holds, the common value is denoted $\int_a^b f(x) \, dx$ and called the **Riemann integral** of $f$ over $[a,b]$. We write $f \in \mathcal{R}[a,b]$ to indicate integrability.
</Definition>

<Theorem title="Integrability criterion" number="6.3">
A bounded function $f: [a,b] \to \mathbb{R}$ is Riemann integrable if and only if for every $\varepsilon > 0$, there exists a partition $P$ such that

$$U(f, P) - L(f, P) < \varepsilon.$$
</Theorem>

<Example title="f(x) = c (constant function)">
Let $f(x) = c$ on $[a,b]$. For any partition $P$, $M_i = m_i = c$, so

$$U(f,P) = c(b-a) = L(f,P).$$

Thus $U - L = 0 < \varepsilon$ for all $\varepsilon > 0$, and $\int_a^b c \, dx = c(b-a)$.
</Example>

<Example title="f(x) = x on [0,1]">
We showed that $U(f, P_n) - L(f, P_n) = \frac{n+1}{2n} - \frac{n-1}{2n} = \frac{1}{n}$, which can be made smaller than any $\varepsilon > 0$. So $f(x) = x$ is integrable on $[0,1]$ with $\int_0^1 x \, dx = \frac{1}{2}$.
</Example>

<Example title="f(x) = x² on [0,1]">
With the uniform partition, $U(f, P_n) - L(f, P_n) = \frac{(n+1)(2n+1) - (n-1)(2n-1)}{6n^2} = \frac{6n}{6n^2} = \frac{1}{n} \to 0$.

So $f(x) = x^2$ is integrable on $[0,1]$ with $\int_0^1 x^2 \, dx = \frac{1}{3}$.
</Example>

## Classes of integrable functions

<Theorem title="Continuous functions are integrable" number="6.4">
If $f: [a,b] \to \mathbb{R}$ is continuous, then $f \in \mathcal{R}[a,b]$.
</Theorem>

<Example title="All elementary continuous functions are integrable">
The functions $\sin x$, $\cos x$, $e^x$, $\ln x$ (on $[a,b] \subseteq (0,\infty)$), $x^n$, and $|x|$ are all continuous on appropriate closed intervals, hence Riemann integrable. For instance, $\int_0^\pi \sin x \, dx = 2$.
</Example>

<Theorem title="Monotone functions are integrable" number="6.5">
If $f: [a,b] \to \mathbb{R}$ is monotone, then $f \in \mathcal{R}[a,b]$.
</Theorem>

<Example title="Monotone integrable function with discontinuities">
The function $f: [0,1] \to \mathbb{R}$ defined by $f(x) = \lfloor nx \rfloor / n$ (floor function scaled) is monotone increasing and has jump discontinuities at $x = k/n$. By the monotone integrability theorem, $f$ is still Riemann integrable.
</Example>

<Theorem title="Bounded functions with finitely many discontinuities" number="6.6">
If $f: [a,b] \to \mathbb{R}$ is bounded and continuous except at finitely many points, then $f \in \mathcal{R}[a,b]$.
</Theorem>

<Example title="Step functions and piecewise continuous functions">
The function $f: [0,3] \to \mathbb{R}$ with $f(x) = 1$ for $0 \leq x < 1$, $f(x) = 2$ for $1 \leq x < 2$, $f(x) = 0$ for $2 \leq x \leq 3$ has two discontinuities (at $x=1$ and $x=2$). It is integrable:

$$\int_0^3 f(x) \, dx = 1 \cdot 1 + 2 \cdot 1 + 0 \cdot 1 = 3.$$
</Example>

## Properties of the integral

<Theorem title="Linearity" number="6.7">
If $f, g \in \mathcal{R}[a,b]$ and $\alpha, \beta \in \mathbb{R}$, then $\alpha f + \beta g \in \mathcal{R}[a,b]$ and

$$\int_a^b (\alpha f + \beta g) = \alpha \int_a^b f + \beta \int_a^b g.$$
</Theorem>

<Example title="Linearity in action">
$\int_0^1 (3x^2 + 5x - 2) \, dx = 3 \int_0^1 x^2 \, dx + 5 \int_0^1 x \, dx - 2\int_0^1 1 \, dx = 3 \cdot \frac{1}{3} + 5 \cdot \frac{1}{2} - 2 \cdot 1 = 1 + \frac{5}{2} - 2 = \frac{3}{2}.$
</Example>

<Theorem title="Order preservation" number="6.8">
If $f, g \in \mathcal{R}[a,b]$ and $f(x) \leq g(x)$ for all $x \in [a,b]$, then

$$\int_a^b f \leq \int_a^b g.$$
</Theorem>

<Example title="Bounding an integral">
Since $0 \leq \sin x \leq 1$ on $[0, \pi/2]$:

$$0 = \int_0^{\pi/2} 0 \, dx \leq \int_0^{\pi/2} \sin x \, dx \leq \int_0^{\pi/2} 1 \, dx = \frac{\pi}{2}.$$

The actual value is $\int_0^{\pi/2} \sin x \, dx = 1$.
</Example>

<Theorem title="Additivity over intervals" number="6.9">
If $f \in \mathcal{R}[a,b]$ and $c \in (a,b)$, then $f \in \mathcal{R}[a,c]$ and $f \in \mathcal{R}[c,b]$, and

$$\int_a^b f = \int_a^c f + \int_c^b f.$$
</Theorem>

<Example title="Splitting an integral">
$\int_0^2 |x - 1| \, dx = \int_0^1 (1-x) \, dx + \int_1^2 (x-1) \, dx = \left[x - \frac{x^2}{2}\right]_0^1 + \left[\frac{x^2}{2} - x\right]_1^2 = \frac{1}{2} + \frac{1}{2} = 1.$
</Example>

<Theorem title="Triangle inequality for integrals" number="6.10">
If $f \in \mathcal{R}[a,b]$, then $|f| \in \mathcal{R}[a,b]$ and

$$\left|\int_a^b f\right| \leq \int_a^b |f|.$$
</Theorem>

<Example title="Triangle inequality example">
$\left|\int_0^{2\pi} \sin x \, dx\right| = |0| = 0$, while $\int_0^{2\pi} |\sin x| \, dx = 4$. Indeed $0 \leq 4$.
</Example>

## Products and compositions

<Theorem title="Product of integrable functions" number="6.11">
If $f, g \in \mathcal{R}[a,b]$, then $fg \in \mathcal{R}[a,b]$.
</Theorem>

<Theorem title="Composition with continuous functions" number="6.12">
If $f \in \mathcal{R}[a,b]$ and $\varphi: \mathbb{R} \to \mathbb{R}$ is continuous, then $\varphi \circ f \in \mathcal{R}[a,b]$.
</Theorem>

<Example title="Composition preserves integrability">
If $f \in \mathcal{R}[a,b]$, then $f^2 = f \cdot f \in \mathcal{R}[a,b]$ (product rule), $|f| \in \mathcal{R}[a,b]$ (take $\varphi(t) = |t|$), $e^f \in \mathcal{R}[a,b]$ (take $\varphi(t) = e^t$), and $\sqrt{|f|} \in \mathcal{R}[a,b]$ (take $\varphi(t) = \sqrt{|t|}$).
</Example>

## The Lebesgue criterion

<Theorem title="Lebesgue criterion for Riemann integrability" number="6.13">
A bounded function $f: [a,b] \to \mathbb{R}$ is Riemann integrable if and only if the set of discontinuities of $f$ has **Lebesgue measure zero**.
</Theorem>

<Example title="Functions with measure-zero discontinuity sets">
- The function $f(x) = \lfloor 1/x \rfloor$ on $[0,1]$ (with $f(0) = 0$) has discontinuities at $x = 1/n$ for $n = 1, 2, 3, \ldots$, a countable set. Countable sets have measure zero, so $f$ is integrable.

- The Thomae function $f: [0,1] \to \mathbb{R}$, defined by $f(p/q) = 1/q$ (in lowest terms) for $x \in \mathbb{Q}$ and $f(x) = 0$ for $x \notin \mathbb{Q}$, is continuous at every irrational and discontinuous at every rational. The rationals in $[0,1]$ have measure zero, so $f$ is Riemann integrable with $\int_0^1 f = 0$.

- The Dirichlet function is discontinuous everywhere. The set $[0,1]$ does not have measure zero, so the function is not Riemann integrable.
</Example>

<Remark>
The Lebesgue criterion reveals the precise limitation of the Riemann integral: it cannot handle functions whose discontinuities form a "large" set. This motivates the Lebesgue integral, which integrates a much larger class of functions.
</Remark>
