# Contraction Mapping Theorem

The Contraction Mapping Theorem (Banach Fixed Point Theorem) guarantees the existence and uniqueness of fixed points for contraction mappings on complete metric spaces. It is one of the most powerful and widely applied results in analysis.

---

## Statement

<Definition title="Contraction mapping" number="9.16">
Let $(X, d)$ be a metric space. A function $T: X \to X$ is a **contraction** (or **contraction mapping**) if there exists a constant $0 \leq q < 1$ such that

$$d(T(x), T(y)) \leq q \cdot d(x, y) \quad \text{for all } x, y \in X.$$

The constant $q$ is called the **contraction factor** (or Lipschitz constant).
</Definition>

<Theorem title="Banach fixed point theorem" number="9.15">
Let $(X, d)$ be a **complete** metric space and $T: X \to X$ a contraction with factor $q < 1$. Then:

1. $T$ has a **unique** fixed point $x^* \in X$ (i.e., $T(x^*) = x^*$).
2. For any starting point $x_0 \in X$, the iterates $x_n = T(x_{n-1}) = T^n(x_0)$ converge to $x^*$.
3. The rate of convergence is $d(x_n, x^*) \leq \frac{q^n}{1 - q} d(x_0, x_1)$.
</Theorem>

<Proof>
**Existence**: Start with any $x_0 \in X$ and define $x_n = T(x_{n-1})$. Then

$$d(x_{n+1}, x_n) = d(T(x_n), T(x_{n-1})) \leq q \cdot d(x_n, x_{n-1}) \leq q^n d(x_1, x_0).$$

For $m > n$:

$$d(x_m, x_n) \leq \sum_{k=n}^{m-1} d(x_{k+1}, x_k) \leq d(x_1, x_0) \sum_{k=n}^{m-1} q^k \leq \frac{q^n}{1-q} d(x_1, x_0) \to 0.$$

So $(x_n)$ is Cauchy. By completeness, $x_n \to x^*$ for some $x^* \in X$.

Since $T$ is continuous (it is Lipschitz), $T(x^*) = T(\lim x_n) = \lim T(x_n) = \lim x_{n+1} = x^*$.

**Uniqueness**: If $T(x^*) = x^*$ and $T(y^*) = y^*$, then $d(x^*, y^*) = d(T(x^*), T(y^*)) \leq q \cdot d(x^*, y^*)$. Since $q < 1$, this forces $d(x^*, y^*) = 0$, so $x^* = y^*$.

**Error bound**: $d(x_n, x^*) = \lim_{m \to \infty} d(x_n, x_m) \leq \frac{q^n}{1-q} d(x_1, x_0)$.
</Proof>

## Applications to equations

<Example title="Solving x = cos(x)">
Consider $T(x) = \cos(x)$ on $[0, 1]$. $|T'(x)| = |\sin x| \leq \sin 1 \approx 0.841 < 1$. By the Mean Value Theorem, $|T(x) - T(y)| \leq 0.841 |x-y|$, so $T$ is a contraction on $[0,1]$.

Starting from $x_0 = 0$: $x_1 = 1$, $x_2 = \cos(1) \approx 0.5403$, $x_3 \approx 0.8576$, ... converging to $x^* \approx 0.7391$, the unique solution of $x = \cos x$.
</Example>

<Example title="Solving x = (x² + 2)/3">
$T(x) = (x^2 + 2)/3$ on $[0, 1]$. $T'(x) = 2x/3$, so $|T'(x)| \leq 2/3 < 1$ on $[0,1]$. Also $T([0,1]) \subseteq [2/3, 1] \subseteq [0,1]$.

$T$ is a contraction on the complete space $[0,1]$. The fixed point satisfies $3x = x^2 + 2$, i.e., $x^2 - 3x + 2 = 0$, giving $x = 1$ or $x = 2$. Since we work on $[0,1]$, the fixed point is $x^* = 1$.

Starting from $x_0 = 0$: $x_1 = 2/3$, $x_2 = 22/27 \approx 0.815$, ... converging to $1$.
</Example>

<Example title="Square root via contraction">
To compute $\sqrt{a}$ for $a > 0$: define $T(x) = \frac{1}{2}(x + a/x)$ (Newton's method). On $[\sqrt{a}, \infty)$:

$$|T'(x)| = \frac{1}{2}\left|1 - \frac{a}{x^2}\right| \leq \frac{1}{2}\left(1 - \frac{a}{x^2}\right) < \frac{1}{2}$$

for $x \geq \sqrt{a}$. So $T$ is a contraction with $q = 1/2$, converging to $\sqrt{a}$.

For $a = 2$, $x_0 = 2$: $x_1 = 3/2 = 1.5$, $x_2 = 17/12 \approx 1.4167$, $x_3 \approx 1.41422$, ... rapid convergence to $\sqrt{2}$.
</Example>

## Applications to differential equations

<Example title="Picard-Lindelof existence theorem">
The initial value problem $y' = f(t, y)$, $y(t_0) = y_0$ can be rewritten as an integral equation:

$$y(t) = y_0 + \int_{t_0}^t f(s, y(s)) \, ds =: (Ty)(t).$$

If $f$ is Lipschitz in $y$: $|f(t, u) - f(t, v)| \leq L|u - v|$, then $T$ is a contraction on $C[t_0 - \delta, t_0 + \delta]$ (with sup metric) for $\delta$ small enough (specifically $\delta < 1/L$).

By the contraction mapping theorem, $T$ has a unique fixed point, which is the unique solution to the ODE. This proves the **Picard-Lindelof theorem**.
</Example>

<Example title="Picard iteration for y' = y, y(0) = 1">
$T(y)(t) = 1 + \int_0^t y(s)\,ds$.

$y_0(t) = 1$, $y_1(t) = 1 + t$, $y_2(t) = 1 + t + t^2/2$, $y_3(t) = 1 + t + t^2/2 + t^3/6$, ...

$y_n(t) = \sum_{k=0}^n \frac{t^k}{k!} \to e^t$. The Picard iterates converge to the exact solution.
</Example>

## Applications to integral equations

<Example title="Fredholm integral equation">
Consider $f(x) = g(x) + \lambda \int_0^1 K(x,t) f(t) \, dt$ where $K$ is continuous and $g$ is given.

Define $(Tf)(x) = g(x) + \lambda \int_0^1 K(x,t) f(t) \, dt$. Then

$$\|Tf_1 - Tf_2\|_\infty \leq |\lambda| \cdot M \cdot \|f_1 - f_2\|_\infty$$

where $M = \sup |K(x,t)|$. So $T$ is a contraction when $|\lambda| < 1/M$, and the equation has a unique solution.
</Example>

## Limitations and extensions

<Example title="Contraction condition is essential">
$T: \mathbb{R} \to \mathbb{R}$, $T(x) = x + 1$ satisfies $|T(x) - T(y)| = |x - y|$ (contraction factor $q = 1$, not $< 1$). No fixed point exists. The strict inequality $q < 1$ is essential.
</Example>

<Example title="Completeness is essential">
$T: (0,1) \to (0,1)$, $T(x) = x/2$. This is a contraction with $q = 1/2$. But $(0,1)$ is not complete, and the "fixed point" $x^* = 0 \notin (0,1)$.
</Example>

<Example title="Functions that are not contractions but have unique fixed points">
$T(x) = x - x^3$ on $[-1/2, 1/2]$ has $T'(x) = 1 - 3x^2$. At $x = 0$: $T'(0) = 1$, so $T$ is NOT a contraction near $0$. Yet $x^* = 0$ is a fixed point. The contraction mapping theorem is sufficient but not necessary for fixed points.
</Example>

<Example title="Solving a system of linear equations iteratively">
Consider $x = 0.1x + 0.2y + 1$, $y = 0.3x + 0.1y + 2$ as $T(x,y) = (0.1x + 0.2y + 1, \, 0.3x + 0.1y + 2)$ on $\mathbb{R}^2$. Using the max metric, $\|T(u) - T(v)\|_\infty \leq 0.4 \|u - v\|_\infty$, so $T$ is a contraction with $q = 0.4$. The unique fixed point is the unique solution to the system.
</Example>

<Example title="Contraction on a function space">
$T: C[0,1] \to C[0,1]$ defined by $(Tf)(x) = \frac{1}{2}\sin(f(x)) + x$. Since $|\sin u - \sin v| \leq |u - v|$, we get $\|Tf - Tg\|_\infty \leq \frac{1}{2}\|f - g\|_\infty$. So $T$ is a contraction with $q = 1/2$ on the complete space $(C[0,1], \|\cdot\|_\infty)$, and there exists a unique continuous function $f$ with $f(x) = \frac{1}{2}\sin(f(x)) + x$.
</Example>

<Example title="Error estimation">
For $T(x) = \cos x$ with $q \approx 0.841$ and $x_0 = 0$: $d(x_0, x_1) = |0 - \cos 0| = 1$. The error after $n$ iterates is at most $\frac{0.841^n}{1 - 0.841} \approx 6.29 \cdot 0.841^n$. After $n = 20$ iterates: error $\leq 6.29 \cdot 0.841^{20} \approx 0.019$, so we have about $2$ decimal places of accuracy.
</Example>

<Remark>
The Banach fixed point theorem is remarkable for its simplicity, power, and wide applicability. It provides existence, uniqueness, and a constructive approximation scheme all at once. Its generalizations (to non-expansive maps, to multi-valued maps, etc.) form a rich branch of nonlinear analysis.
</Remark>
