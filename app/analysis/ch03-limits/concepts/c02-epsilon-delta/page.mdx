# 엡실론-델타 정의 (Epsilon-Delta Definition)

The epsilon-delta formalism is the language of rigorous analysis. Mastering this technique is the single most important skill in a first course on real analysis. Here we develop fluency through systematic examples.

---

## The structure of an epsilon-delta proof

<Remark>
To prove $\lim_{x \to c} f(x) = L$, we follow this template:

1. **Let** $\varepsilon > 0$ be given. (This is the "for all" quantifier.)
2. **Choose** $\delta > 0$ (depending on $\varepsilon$). (This is the "there exists" quantifier.)
3. **Verify** that $0 < |x - c| < \delta$ implies $|f(x) - L| < \varepsilon$.

The art lies in Step 2: finding the right $\delta$. Often we first do **scratch work** to find $\delta$, then write the clean proof.
</Remark>

---

## Linear functions

<Example title="Linear function">
1. **Prove**: $\lim_{x \to 2} (5x - 3) = 7$.

**Scratch work**: $|5x - 3 - 7| = 5|x - 2|$. We need $5|x-2| < \varepsilon$, so $|x-2| < \varepsilon/5$.

**Proof**: Let $\varepsilon > 0$. Choose $\delta = \varepsilon/5$. If $0 < |x - 2| < \delta$, then

$$|5x - 3 - 7| = 5|x - 2| < 5\delta = \varepsilon. \qquad \square$$
</Example>

---

## Quadratic functions

<Example title="Quadratic at a point">
2. **Prove**: $\lim_{x \to 1} x^2 = 1$.

**Scratch work**: $|x^2 - 1| = |x-1| \cdot |x+1|$. We need to bound $|x+1|$. If $|x - 1| < 1$, then $0 < x < 2$, so $|x+1| < 3$. Thus $|x^2 - 1| < 3|x-1|$.

**Proof**: Let $\varepsilon > 0$. Choose $\delta = \min(1, \varepsilon/3)$. If $0 < |x-1| < \delta$, then $|x+1| < 3$ and

$$|x^2 - 1| = |x-1| \cdot |x+1| < \delta \cdot 3 \leq \varepsilon. \qquad \square$$
</Example>

<Example title="Quadratic at zero">
3. **Prove**: $\lim_{x \to 0} x^2 = 0$.

**Proof**: Let $\varepsilon > 0$. Choose $\delta = \sqrt{\varepsilon}$. If $0 < |x| < \delta$, then $|x^2| = x^2 < \delta^2 = \varepsilon$.
</Example>

---

## Rational functions

<Example title="Rational function limit">
4. **Prove**: $\lim_{x \to 2} \frac{1}{x} = \frac{1}{2}$.

**Scratch work**: $\left|\frac{1}{x} - \frac{1}{2}\right| = \frac{|x - 2|}{2|x|}$. We need a lower bound on $|x|$. If $|x - 2| < 1$, then $x > 1$, so $|x| > 1$ and $\frac{1}{|x|} < 1$.

**Proof**: Let $\varepsilon > 0$. Choose $\delta = \min(1, 2\varepsilon)$. If $0 < |x - 2| < \delta$, then $|x| > 1$ and

$$\left|\frac{1}{x} - \frac{1}{2}\right| = \frac{|x-2|}{2|x|} < \frac{\delta}{2} \leq \varepsilon. \qquad \square$$
</Example>

<Example title="More complex rational function">
5. **Prove**: $\lim_{x \to 3} \frac{x^2 - 9}{x - 3} = 6$.

Note $\frac{x^2 - 9}{x - 3} = x + 3$ for $x \neq 3$. So $|f(x) - 6| = |x + 3 - 6| = |x - 3|$.

**Proof**: Let $\varepsilon > 0$. Choose $\delta = \varepsilon$. If $0 < |x - 3| < \delta$, then $\left|\frac{x^2 - 9}{x - 3} - 6\right| = |x - 3| < \delta = \varepsilon$.
</Example>

---

## Square root functions

<Example title="Square root limit">
6. **Prove**: $\lim_{x \to 4} \sqrt{x} = 2$.

**Scratch work**: $|\sqrt{x} - 2| = \frac{|x - 4|}{|\sqrt{x} + 2|}$. Since $\sqrt{x} + 2 \geq 2$ for $x \geq 0$, we have $|\sqrt{x} - 2| \leq \frac{|x-4|}{2}$.

**Proof**: Let $\varepsilon > 0$. Choose $\delta = 2\varepsilon$. If $0 < |x - 4| < \delta$ (with $x > 0$), then

$$|\sqrt{x} - 2| = \frac{|x-4|}{\sqrt{x}+2} \leq \frac{|x-4|}{2} < \frac{\delta}{2} = \varepsilon. \qquad \square$$
</Example>

---

## Trigonometric functions

<Example title="Sine limit">
7. **Prove**: $\lim_{x \to 0} \sin x = 0$.

Using $|\sin x| \leq |x|$: let $\varepsilon > 0$, choose $\delta = \varepsilon$. If $0 < |x| < \delta$, then $|\sin x - 0| \leq |x| < \varepsilon$.
</Example>

<Example title="x sin(1/x) limit">
8. **Prove**: $\lim_{x \to 0} x\sin(1/x) = 0$.

$|x\sin(1/x)| \leq |x| \cdot 1 = |x|$. Choose $\delta = \varepsilon$.
</Example>

---

## Negation: proving limits do not exist

<Definition title="Negation of limit existence" number="3.6">
$\lim_{x \to c} f(x) \neq L$ means:

$$\exists \varepsilon_0 > 0, \quad \forall \delta > 0, \quad \exists x : \quad 0 < |x - c| < \delta \text{ and } |f(x) - L| \geq \varepsilon_0.$$

The limit does not exist at $c$ if this holds for every $L \in \mathbb{R}$.
</Definition>

<Example title="Topologist sine does not have a limit">
9. $\lim_{x \to 0} \sin(1/x)$ does not exist.

For any candidate $L$ and any $\delta > 0$: in $(0, \delta)$, $\sin(1/x)$ takes the values $1$ and $-1$ (at $x = \frac{1}{\pi/2 + 2\pi k}$ and $x = \frac{1}{-\pi/2 + 2\pi k}$). So $|f(x) - L| \geq 1$ for either $f(x) = 1$ or $f(x) = -1$. Take $\varepsilon_0 = 1/2$.
</Example>

<Example title="Dirichlet function has no limit anywhere">
10. Define $f(x) = 1$ if $x \in \mathbb{Q}$, $f(x) = 0$ if $x \notin \mathbb{Q}$. For any $c$ and any $\delta > 0$, both rationals and irrationals exist in $(c - \delta, c + \delta) \setminus \{c\}$. So $f$ takes both values $0$ and $1$ near $c$, and the limit cannot exist.
</Example>

## Common pitfalls

<Example title="The delta depends on epsilon">
11. A common error is choosing $\delta$ independent of $\varepsilon$. The statement requires that for each $\varepsilon$ there exists a suitable $\delta$. Usually $\delta$ decreases as $\varepsilon$ decreases. For $\lim_{x \to 0} x^2 = 0$, we need $\delta = \sqrt{\varepsilon}$, which depends on $\varepsilon$.
</Example>

<Example title="The min trick">
12. When bounding $|f(x) - L|$, we often need to restrict $|x - c|$ first (e.g., $|x - c| < 1$) to control a factor. This leads to $\delta = \min(1, g(\varepsilon))$ for some function $g$. The choice of $1$ is arbitrary; any fixed positive constant works. The key is that $\delta$ must simultaneously satisfy both constraints.
</Example>

## The epsilon-delta method as a game

<Remark>
Think of the epsilon-delta definition as a two-player game:
- **Adversary** chooses $\varepsilon > 0$ (the challenge).
- **Prover** responds with $\delta > 0$ (the strategy).
- The limit exists if the prover has a **winning strategy** for every challenge.

This game-theoretic perspective makes the quantifier structure intuitive: the adversary goes first ($\forall \varepsilon$), then the prover responds ($\exists \delta$), and the claim must hold for all $x$ in the resulting region ($\forall x$).
</Remark>
